Graphic Design

4K Walk 1K Run Kids Dash

This event poster uses bright color, playful illustration, and oversized type to create an energetic community-run identity. The running characters and sun motif make the event feel active and family friendly.

The layout balances excitement with practical information by separating the registration area, sponsor logos, and event details into clear zones. It is designed to catch attention while still being useful at a glance.
